# GitHub Connect checklist — Android phone

Ye checklist tab use karo jab Lovable ka **GitHub → Connect project / Add connection** button phone pe blank ho, kuch open na kare, ya stuck rahe.

## Fast tries

1. Chrome menu (⋮) → **Desktop site** ON karo, phir Lovable reload karke GitHub connect retry karo.
2. Chrome → Settings → Site settings → **Pop-ups and redirects** → Allowed.
3. Chrome → Settings → Site settings → **Third-party cookies** → Allow.
4. Lovable tab close karo, Chrome recent apps se bhi close karo, phir dobara open karo.
5. Same flow **Incognito tab** me try karo.

## Site data reset

1. Chrome me `lovable.dev` kholo.
2. Address bar ke left lock/info icon pe tap karo.
3. Site settings → **Clear data**.
4. Lovable me dobara login karo.
5. GitHub → Connect project retry karo.

## Alternate browser

Agar Chrome block kar raha hai, ek baar ye browser try karo:

- Firefox for Android
- Samsung Internet
- Microsoft Edge

Lovable me login karo → project kholo → GitHub → Connect project.

## Agar fir bhi nahi hota

`.lovable/support-message.md` ka message copy karke Lovable support ko bhejo. Ye account-level OAuth popup issue hai; AI isko bypass nahi kar sakta.
