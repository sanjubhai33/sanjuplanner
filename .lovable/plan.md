## Plan: "Edited with Lovable" badge hataana

Aapke workspace ke paid plan ka use karke, main published site se **"Edited with Lovable"** badge ko hide kar dunga.

### Kya hoga
- `set_badge_visibility` tool ko `hide_badge: true` ke saath call karunga
- Agar workspace ka plan Pro ya usse upar hai, badge turant hide ho jayega (agli publish/deploy ke baad live site par nahi dikhega)
- Agar plan eligible nahi nikla, tool error dega aur main aapko bata dunga

### Aapka action
Bas is plan ko **Approve** karo — main badge hide kar dunga.
